# Lab 05

## Co zostało zrealizowane
Skrypt w Pythonie (`bpy`) generujący las z różnymi typami roślin (drzewo, krzew, paproć). Zaimplementowano logikę biomów, która rozmieszcza rośliny względem centrum sceny, organizację obiektów w kolekcjach Blendera oraz proceduralne podłoże. Skrypt automatycznie renderuje gotową scenę.

## Uruchomienie
Skrypt uruchamia się z poziomu terminala lub w edytorze Blendera. 
Gotowy render (`las_05.png`) zapisze się automatycznie w tym samym folderze.
