# Lab04

## Co zostało zrealizowane
Skrypt w Pythonie wykorzystujący Blender API (`bpy`) do proceduralnego generowania sceny 3D. Skrypt tworzy funkcję budującą rośliny z prymitywów (walce i kostki), używając trygonometrii do rozmieszczenia liści i korzeni. Skrypt przypisuje proceduralne materiały, ustawia oświetlenie oraz kamerę, a na koniec renderuje scenę z trzema różnymi roślinami do pliku PNG.

## Uruchomienie
Skrypt uruchamia się z poziomu terminala.

Gotowy render PNG zapisze się automatycznie w tym samym folderze.
