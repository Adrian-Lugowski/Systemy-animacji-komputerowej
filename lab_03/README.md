# Lab 03

## Co zostało zrealizowane
Zaimplementowano symulację pokazu fajerwerków przy użyciu Canvas API. W projekcie znajudują się trzy klasy (`Particle`, `Firework`, `FireworkShow`). Dodano automatyczne odpalanie rakiet oraz suwaki do zmiany grawitacji i liczby cząstek.

## Uruchomienie
Otwórz plik `index.html` w dowolnej przeglądarce.
